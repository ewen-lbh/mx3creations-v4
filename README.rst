Build Setup
===========

.. code ::

    # install dependencies
    $ yarn install

    # serve with hot reload at localhost:3000
    $ yarn dev

    # build for production and launch server
    $ yarn build
    $ yarn start

For detailed explanation on how things work, check out `Nuxt.js
docs <https://nuxtjs.org>`__.
